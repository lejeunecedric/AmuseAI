## API Models (from Amuse.API)

**Text2ImgRequest:**
- Prompt (required, string)
- NegativePrompt (optional, string)
- Width (512-1024, default 512)
- Height (512-1024, default 512)
- Steps (1-100, default 20)
- GuidanceScale (0.0-20.0, default 7.5)
- Seed (optional, int)

**Img2ImgRequest:**
- Prompt (required, string)
- NegativePrompt (optional, string)
- Image (required, Base64 string)
- Strength (0.0-1.0, default 0.75)
- Width (512-1024, default 512)
- Height (512-1024, default 512)
- Steps (1-100, default 20)
- GuidanceScale (0.0-20.0, default 7.5)
- Seed (optional, int)

**UpscaleRequest:**
- Image (required, Base64 string)
- Scale (2 or 4, default 2)
- TileMode (boolean, default false)

## API Endpoints

- POST /api/generate/text2img → Returns job ID
- POST /api/generate/img2img → Returns job ID
- POST /api/upscale → Returns job ID

All endpoints return: `{ jobId, status, checkStatusAt }`
